# Changelog

## 1.0.0

- Initial release
- Death messages for all 16 causes of death
- Monster-specific kill messages for 21+ monsters
- Non-lethal monster encounter messages with cooldowns
- Event messages (critical damage, ship leaving, vote to leave, teleporter, quota)
- Modded/custom enemy support with fallback messages
- Configurable enemy blacklist
- Extended chat message duration
